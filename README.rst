A* in 3D fun
-----------------------------------

Just something I did a few years ago; uses OpenGL with GLUT.

Buttons:

-	TAB: toggle showing the paths being calculated

-	Z: zoom out

- z: zoom in

- r/R: rotate

- space: start/stop walking

- mouse: click and drag to reposition target marker
